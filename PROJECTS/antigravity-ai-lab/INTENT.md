# PROJECT INTENT: Antigravity AI Lab

## 🎯 Goal
Acquire Google Cloud for Startups approval by demonstrating a professional, scalable, and technically deep AGI ecosystem.

## 👥 Target Persona
- **Google Cloud Reviewers**: Need to see infrastructure depth (Next.js, Cloud Run, Postgres).
- **Venture Capitalists/Investors**: Need to see a vision for the "Operating System for Agents."
- **Enterprise Partners**: High-grade stability and security (MessagePack, Auth-First).

## 🛠 Use Case
- Presenting the **Antigravity AGI Engine** as a modular framework.
- Showcasing each project (Gniewka, WAR, SYF...) as a production-grade module.

## 🎭 Mood & Atmosphere
- **Professional AGI**: Modern, dark, luminescent.
- **High-Fidelity**: 4x100 Lighthouse performance, pixel-perfect layout.
- **Trustworthy**: Secure, standalone, production-ready.
- **Futuristic**: Space Grotesk typography, subtle CRT and glitch accents (limited for readability).

## 🧱 Constraints
- **Stack**: Next.js 16, React 19, Tailwind CSS v4, Framer Motion.
- **Language**: 100% Technical English.
- **Aesthetics**: Premium/High-End. No cheap art-manifesto vibes for the `/tech` section.
