## 📜 10-ATOMÓWKA Hard Rules
1. **NO PLACEHOLDERS**: Only AI-generated assets or technical readouts.
2. **KINETIC ENTROPY**: Every element must have a secondary motion state (Glitch/Pulse).
3. **GRID SUPREMACY**: Align everything to a sub-pixel grid system.
4. **HAPTIC TEXTURE**: Use Halftones and Dithering to create visual "grit."
5. **DENSITY MAXIMA**: If a pixel is empty, fill it with a diagnostic coordinate.
