### POLSKIE SŁOWNICTWO TYPOGRAFICZNE

| Termin | Znaczenie | Jak naprawić |
|--------|-----------|--------------|
| **SIEROTA** | Pierwszy wiersz akapitu sam na dole strony/kolumny | CSS `orphans: 2` |

### SIEROTY - WSZYSTKO CO MUSISZ WIEDZIEĆ

#### 1. Spójniki na końcu linii (NAJCZĘSTSZE)

**PROBLEM:**
```
Idę do sklepu i
kupię chleb.
```

**ROZWIĄZANIE CSS:**
```css
/* Nowoczesne - Chrome 114+, Firefox 121+ */
p, li, span {
    text-wrap: pretty;
    text-wrap: balance; /* dla krótkich tekstów */
}

/* Fallback */
.no-orphan {
    word-spacing: 0.05em;
}
```

**ROZWIĄZANIE JS (najpewniejsze):**
```javascript
function fixPolishOrphans(text) {
    // Lista polskich spójników i przyimków jednoliterowych
    const conjunctions = [
        'i', 'w', 'z', 'a', 'o', 'u', 'e',  // jednoliterowe
        'do', 'na', 'po', 'od', 'za', 'ze', 'we', // dwuliterowe
        'lub', 'nie', 'ale', 'czy', 'jak', 'tak', // trzyliterowe
        'oraz', 'albo', 'więc', 'tylko'  // czteroliterowe
    ];
    
    // Regex dla wszystkich
    const regex = new RegExp(
        `\\s(${conjunctions.join('|')})\\s`,
        'gi'
    );
    
    // Zamień spację po spójniku na &nbsp;
    return text.replace(regex, ' $1\u00A0');
}

// Użycie
document.querySelectorAll('p, li, h1, h2, h3').forEach(el => {
    el.innerHTML = fixPolishOrphans(el.innerHTML);
});
```

**ROZWIĄZANIE HTML (manualne):**
```html
<!-- Zamiast -->
<p>Idę do sklepu i kupię chleb.</p>

<!-- Rób -->
<p>Idę do sklepu i&nbsp;kupię chleb.</p>

<!-- Lub używaj <span> z nowrap -->
<p>Idę do sklepu <span class="nowrap">i kupię</span> chleb.</p>
```

```css
.nowrap {
    white-space: nowrap;
}
```

### JAVASCRIPT UTILITY LIBRARY

```javascript
/**
 * POLISH TYPOGRAPHY FIXER
 * Naprawia sieroty, wdowy, bękarty w polskim tekście
 */
const PolishTypography = {
    // Spójniki i przyimki do ochrony
    PROTECTED_WORDS: [
        // 1 litera
        'a', 'i', 'o', 'u', 'w', 'z', 'e',
        // 2 litery
        'do', 'na', 'po', 'od', 'za', 'ze', 'we', 'że', 'by', 'bo', 'to', 'co', 
        'ni', 'no', 'ta', 'te', 'tu', 'ty',
        // 3 litery
        'ale', 'lub', 'nie', 'czy', 'jak', 'tak', 'już', 'też', 'pod', 'nad', 
        'bez', 'dla', 'przy', 'oraz', 'gdy',
        // 4 litery  
        'albo', 'więc', 'oraz', 'tylko', 'może', 'gdyż', 'jako', 'przy'
    ],

    // Zamień spacje po chronionych słowach na &nbsp;
    fixOrphans(text) {
        const words = this.PROTECTED_WORDS.join('|');
        const regex = new RegExp(`\\s(${words})\\s`, 'gi');
        return text.replace(regex, ' $1\u00A0');
    },

    // Napraw ostatnie słowo akapitu
    fixWidows(text) {
        return text.replace(/\s(\S{1,6})$/gm, '\u00A0$1');
    },

    // Napraw oba
    fix(text) {
        return this.fixWidows(this.fixOrphans(text));
    },

    // Zastosuj do elementów DOM
    apply(selector = 'p, li, h1, h2, h3, h4, h5, h6, blockquote, figcaption') {
        document.querySelectorAll(selector).forEach(el => {
            // Tylko tekstowe węzły, nie HTML
            el.childNodes.forEach(node => {
                if (node.nodeType === Node.TEXT_NODE) {
                    node.textContent = this.fix(node.textContent);
                }
            });
        });
    },

    // Init przy DOMContentLoaded
    init() {
        if (document.readyState === 'loading') {
            document.addEventListener('DOMContentLoaded', () => this.apply());
        } else {
            this.apply();
        }
    }
};

// Użycie:
// PolishTypography.init();
```

### FLOW TEKSTU

1. **Rytm wertykalny**
   - Konsekwentny `line-height`
   - Marginesy w wielokrotnościach `line-height`
   - Wszystko na siatce

2. **Długość linii**
   - Ideał: 45-75 znaków
   - Max: 80 znaków
   - Na mobile: 35-50 znaków

3. **Hierarchia**
   - H1 > H2 > H3 > body - ZAWSZE
   - Skok max 2 poziomy (nie rób H1 → H4)

4. **Kerning & Tracking**
   - Headlines: tighter (-0.02em do -0.05em)
   - Body: normal
   - Small: looser (+0.01em)

```css
h1, h2, h3 {
    letter-spacing: -0.02em;
}

.small-text {
    letter-spacing: 0.01em;
}
```

### PUSTE LINIE / WHITESPACE

- Między sekcjami: 2-3x wysokość linii
- Między akapitami: 1x wysokość linii
- Po nagłówku: 0.5x wysokość linii
- NIGDY: losowe marginesy

**SKRYPT DO PRZETWARZANIA PLIKU MD:**

```bash
#!/bin/bash
# fix-orphans.sh - napraw sieroty w pliku MD

INPUT="$1"
OUTPUT="${INPUT%.md}_fixed.md"

  s/ ([iwazoueIWAZOUE]) / \1\xC2\xA0/g
```
