# ZASADY OBOWIĄZKOWE

## LINKI:

1. **Minimum 5 linków** dla każdej propozycji
2. **Zawsze Google Images** z konkretną frazą
3. **Zawsze Wikipedia** jeśli istnieje
4. **Pinterest** dla inspiracji wizualnej
5. **Źródła branżowe:** Eye Magazine, It's Nice That, Dezeen, AIGA, Designboom

## FORMAT LINKÓW:

```markdown
- [Czytelna nazwa](pełny_URL)
```

NIE:
```markdown
- pełny_URL
```

## OPISY:

1. **Kim jest** - nie "co robi", ale KIM JEST (tożsamość)
2. **Dlaczego warto** - konkretne powody, nie "jest fajny"
3. **Cytaty** - oryginalny język + tłumaczenie
4. **Potencjał** - ocena dla KONKRETNEGO projektu

## STRUKTURA WIZUALNA:

1. **Separatory** między sekcjami (═══ lub ---)
2. **Emoji** dla szybkiego skanowania (⭐ = ulubieniec)
3. **Bold** dla kluczowych terminów
4. **Checkboxy** dla feedbacku
