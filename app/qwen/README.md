# SYF — 10 Homepage Layouts

## 🎨 Collection of Awwwards-level layouts exploring AI authenticity vs simulation

### Layouts:

1. **THE QUANTUM GATE** - WebGL particle portal with mouse interaction
   - Features: Particle system, gradient text, glass cards
   - Colors: Deep purple + electric blue
   - Hook: Mouse-following particles with connection lines

2. **BROKEN GRID CHAOS** - Asymmetric 16-column grid
   - Features: Intentional misalignment, magnetic hover
   - Colors: Monochrome + neon accent
   - Hook: Cards pull toward cursor on hover

3. **HORIZONTAL TIMELINE** - Left-scroll narrative experience
   - Features: 4 chapters, progress bar, floating shapes
   - Colors: Warm gradient (orange → pink)
   - Hook: Horizontal scroll with parallax

4. **BRUTALIST RAW** - Anti-design statement
   - Features: Monospace fonts, raw borders, harsh transitions
   - Colors: Black + white + shock red
   - Hook: Brutal hover states, no smooth animations

5. **GLASSMORPHISM DEPTH** - Layered transparency
   - Features: Frosted glass cards, animated gradient background
   - Colors: Purple gradient + white
   - Hook: 3D tilt effect on mouse move

6. **TYPOGRAPHY AS ART** - Kinetic type focus
   - Features: Massive lettering, hover effects on characters
   - Colors: White + accent purple
   - Hook: Text scramble animation

7. **SPLIT REALITY** - Dual-world interactive
   - Features: Draggable divider, AI vs Human themes
   - Colors: Cold (AI) vs Warm (Human)
   - Hook: User controls the split

8. **FLUID ORGANIC** - Blob-based organic shapes
   - Features: Morphing blobs, no straight lines
   - Colors: Earth tones
   - Hook: Continuous blob animation

9. **MINIMAL GALLERY** - Museum-style curation
   - Features: One artwork visible, lots of whitespace
   - Colors: Pure white + charcoal
   - Hook: Smooth transitions, minimal UI

10. **MAXIMALIST COLLAGE** - Controlled chaos
    - Features: Moving gradient, floating stickers, shake animation
    - Colors: Rainbow gradient
    - Hook: Everything moves, intentionally overwhelming

## 🎯 Design Principles

- **Pure HTML/CSS/JS** - No frameworks, no dependencies
- **50/10 Level** - Awwwards-quality interactions
- **Thematic** - All explore AI authenticity question
- **Interactive** - Each has unique hook/interaction
- **Responsive** - All layouts work on mobile

## 🚀 Usage

Open `index.html` in browser. Use navigation bar to switch between layouts.
