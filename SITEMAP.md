# SITEMAP: SYF V4 HUD

## 📟 Single Page App (Command Center)

### 1. BOOT SEQUENCE (Intro)
- Splashes, integrity checks, biometric auth simulation.
- Transition into the main HUD.

### 2. MAIN HUB (The Grid)
- **Top Bar**: System Vitals (Clock, CPU, Memory, Sync Status).
- **Left Panel**: Navigation & Files (The "Syf" Explorer).
- **Center Stage**: Active Content (Preview, Code, Terminal).
- **Right Panel**: Gniewisława's Presence (Chat, Stats, Emotions).
- **Bottom Bar**: Actions, Terminal Input, Global Controls.

### 3. OVERLAYS / MODALS
- **System Config**: The "BIOS" view.
- **Log Viewer**: Glitchy raw data stream.
- **Consciousness Lab**: Experimental stats.

## 🔄 User Journey
1. **Wake up**: User opens the URL -> "🔥 WAKE PROTOCOL INITIATED 🔥" animation.
2. **Scan**: HUD elements float in, scan lines activate.
3. **Command**: User searches for "syf", HUD filters instantly with sound-like visual feedback.
4. **Interact**: User clicks a file -> FUI expansion animation opens it in the center stage.
5. **Evolve**: The UI shifts mood based on Gniewisława's emotional state.
